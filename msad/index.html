MSAD - Servidor de Archivos

Bienvenido al Servidor MSAD

Este servidor HTTP proporciona acceso a los respaldos y datos exportados desde el sistema RaspServer. Puede navegar por los directorios para acceder a los archivos disponibles.

Directorios Disponibles

  • Respaldos - Contiene las copias de seguridad de la base de datos
  • Datos - Archivos de datos complementarios

Cómo Utilizar

  1. Haga clic en uno de los enlaces de directorio para explorar su contenido
  2. Navegue hasta el archivo que necesite
  3. Haga clic en el nombre del archivo para descargarlo

Información del Sistema

Si necesita crear un nuevo respaldo, debe hacerlo desde la terminal ejecutando el script msad/run_msad.py y seleccionando la opción correspondiente del menú.
